EasingFunction

public struct EasingFunction<Value> : Hashable where Value : SIMDRepresentable

An easing function powered by a Bezier that can be used with a BasicAnimation.

Note

This can be used on its own, but it’s mainly used by BasicAnimation‘s tick method.
